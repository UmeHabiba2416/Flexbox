## â“ Interview / Viva Ready Answers

**What is Flexbox?**  
Flexbox is a one-dimensional CSS layout system used to align elements efficiently.

**Difference between Flexbox and Grid?**  
Flexbox is 1D (row or column), Grid is 2D (rows and columns).

**Why use flex-wrap?**  
To allow items to move to the next line on smaller screens.

**How does Flexbox help responsiveness?**  
It automatically adjusts layout without fixed sizes.
